load()method 
public function load(accountId:String):void

Loads an interstitial advert to be displayed at a later point. InterstitialEvent.LOADED is dispatched when the interstitial is ready.

The Account or Ad Unit ID used here can be different from the one you used to initialise the platform. This allows you to load a different interstitial in the background to be displayed at an appropriate time in your application.

accountId:String
